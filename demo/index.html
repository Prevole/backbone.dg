Datagrid Demo

Backbone Datagrid is a plugin to organize present data with some controls like sorting, pagging, filtering. Backbone and Backbone.Marionette are used to offer a simple and extensible way to present the data in a structured way. A default styling is proposed through the usage of Twitter Bootstrap. This should be easily overriden by your own styling.

Table with default setup

In most of cases, usual components like simple filter, sorting, pagging are used to present data into a table. Backbone Datagrid offers these components out of the box like shown in the following demo. Every part of the plugin implementation could be overriden to provide a custom way to present data.

Table with duplicate regions on top and bottom

The default components provided by the plugin could be used twice or more on a single data grid. By default, the implementation provided automatically synchronized such views as far as they share some common structure and styling.

Table where toolbar and items per page are removed

When some components are not required to present your datagrid, you can easily configure your grid to remove the components that you do not want to show. This is done by configuration when you create the data grid layout.

Grid with a totally different markup

That is not so unfrequent to present data under a different markup that usual <table /> tag. The plugin offers an easy way to change the layout and the related markup. In the following example, only divs and spans are used to present the data. The other components remains usable as they do not manipulate directly the markup of the data.
