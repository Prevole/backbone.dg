¶ Dg.QuickSearch

Default implementation for the quick search accross the collection. A text field is used to get the search term and the filtering happens on the keyup event from the text field.

The refresh of the collection happens in a delayed function to allow writing more than one caracter before triggering the refresh. This will avoid strange behavior and brings a better user experience.

 
12  Dg.QuickSearchView = Dg.DefaultItemView.extend
13    template: templates['quicksearch']
14  
15    events:
16      'keyup input': 'search'

18    ui:
19      term: 'input'
20  
21    _searchInternal: _.debounce(
22      (event) ->
23        @update _.object( [infoKeys.term], [@ui.term.val().trim()] )
24      , 300
25    )
