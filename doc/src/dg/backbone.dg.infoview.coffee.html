Dg.InfoView

¶ Dg.InfoView

Default implementation to present general information about the collection currently displayed.

The following information are shown:

  • from: The number of the first entry shown
  • to: The number of the last entry shown
  • total: The total number of entries shown

A translation is done for the message shown through I18n-js if present. Otherwise, the default message is used:

  • default message: Showing ${from} to ${to} of ${total} entries
